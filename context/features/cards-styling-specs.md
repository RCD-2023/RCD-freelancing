# Cards styling when hover

## Overview
Add a subtle hover animation to the service cards. On hover, the card should smoothly move upward by a few pixels (lift effect) the top border should transition to the accent color and a box-shadow. Use CSS transitions for a smooth and professional feel.

## Requirements
- Createa a custom property (css variable). My proposal is `--accent-color: #2563eb`; but you can check if other color is more suited to the rest of colors.
- Check to be suited on dark and light
