# Add theme button (Dark/Light)

## Overview

Create a dark/light theme toggle for this site, and make it functional (add js code)

## Requirements

- Use a button element.
- Use localStorage to persist the user's preference.
- Respect the user's system preference on first visit.
- Toggle a `dark-theme` class on the body element.
- Use Font Awesome icons:
  - moon icon for dark mode
  - sun icon for light mode
- Update aria-pressed appropriately.
- Include smooth color transitions.
- Keep the implementation simple and accessible.
- Place all JavaScript in main.js.
- Do not use any frameworks or external dependencies other than Font Awesome.
- For dark mode theme use:
    - Background: `#121212`,
    - Surface: `#1e1e1e`, 
    - Text color adjust accordingly or use: `#e5e5e5`
    - Border: `#333333`
