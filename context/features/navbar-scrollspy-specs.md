# Add Scrollspy effect to navbar

## Overview

I would like the navbar links to have an animated hover underline effect.

When a user hovers over a navigation link, an underline should smoothly animate into view.

In addition, when a navigation link is clicked and becomes the active section, the underline should remain visible as a persistent active navigation indicator.

## Requirements:
- Smooth underline animation on hover.
- The active link should keep its underline even when not hovered.
- Only one navigation link should be active at a time.
- The active state should update as the user navigates through the page sections.
- Reuse the same underline style for both hover and active states.
- Ensure the implementation works well on desktop and mobile.
