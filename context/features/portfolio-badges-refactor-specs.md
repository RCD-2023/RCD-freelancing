# Refactor Portfolio Badges

## Overview
Update the category badges displayed on the portfolio cards.

Use the following categories:

| Romanian | English |
|----------|---------|
| Îmbrăcăminte | Fashion |
| Electronice | Electronics |
| Imobiliare | Real Estate |
| Blog culinar | Food Blog |
| Developer Tool (SaaS) | Developer Tool (SaaS) |
| Aplicație management date | Data Management App |
| Portal medical | Health Portal |

## Requirements
- Update only the portfolio category badges.
- Add or update both Romanian and English translations.
- Keep the existing badge component and styling.
- Do not modify card titles, descriptions, images or links.
- Do not change badge colors, spacing or typography.
- Reuse the existing i18n system and translation keys where possible.

## Acceptance criteria
- Every portfolio card displays the correct category badge.
- Badge text changes correctly when switching between Romanian and English.
- All badges have a consistent visual appearance.
- No layout regressions are introduced.
- No console errors are generated.
